🔥 How To Set Up Firebase (One-Time, 5 Minutes)
1
Create a Firebase Project
Go to console.firebase.google.com → Add Project → Give it a name (e.g. study-app-db) → Continue
2
Enable Firestore Database
Build → Firestore Database → Create Database → Start in Test Mode → Choose region (nam5 recommended) → Done
3
Enable Realtime Database (for presence)
Build → Realtime Database → Create Database → Start in Test Mode → Done
4
Get Your Config Keys
Project Settings (⚙️) → Your Apps → Web App (</>) → Register App → Copy the firebaseConfig object
